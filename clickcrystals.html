About us

The ClickCrystals Project is a free and open-source Minecraft client modification that allows all-in-one features without you having to download a lot of mods.

Is this legit?

While ClickCrystals mainly supports hotkeying, the concept has sparked debates amongst the crystal pvp community. Currently, the mod is banned on most servers, but is not flaggable by anticheat.

However, it is best to prevent being screenshared!

Customizability

ClickCrystals is also highly customizable with over 45 modules, each with a unique set of settings to customize your user experience.

Easy to Manage

There are multiple search bars across the user interface for the mod, facilitating the process of searching for what you need!
