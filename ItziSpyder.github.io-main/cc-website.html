ClickCrystals

ClickCrystals is an free and open-source client modification for Minecraft Java Edition 1.20.x

ClickCrystals has many modules, each containing a unique feature, everything is toggleable and fully customizable!

ClickCrystals works by providing custom pvp features like hotkey support to boost your performance without ruining the vanilla experience, download the mod today @ Modrinth.com!
