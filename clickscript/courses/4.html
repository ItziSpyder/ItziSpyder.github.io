Sending messages

Learn how to send messages to the client, or in global chat as a message or a command.

Direct messaging the client

To send a message to the client player, the command would be send. The argument would be a message.

send Hello World

Sending message to chat

To send a message as the player to the global chat, replace send with say.

say Hi everyone

Executing a command

Executing a command would be the same as sending a global chat message, but the message would start with a /.

say /give @s diamond 64
