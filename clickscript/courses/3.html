Creating Modules

Creating your first module: declaration, setting description, toggling, and reloading!

Declaring a module

Modules are declared with the command module. The argument to pass in for this command would be create.

// module declaration
module create new-module

Setting description

Descriptions are set using the command description. The argument to pass in would be a message of your choice.

// module declaration
module create new-module
description This is a test module

Toggling modules

Modules are toggled using also the module command.

Possible arguments are

  • enable
  • disable
module enable new-module
module disable new-module

Reloading your modules

To reload the module, go in game and type the command

,ccs reload-scripts

When done, click on your apostrophe ' key, then navigate to the Custom Made tab.

Reloading your scripted modules would disable them. You'll have to renable them.
