Looping and Waiting

Learn how to repeat commands instantly, or with a period. Also learn about waiting before executing another command.

Waiting

Waiting can become very useful when you want to wait for a bit before continuing down the command line. The command to wait is just the word wait. The second argument would be the number of seconds to wait, and it can be a decimal.

The third argument would just be the command line to execute.

wait 0.5 ...

The ... can be replaced with any command line.

wait 1.0 send Hello world

Recall from our previous lesson that send would privately dm the client, and not say something in global chat.

Another type of waiting is wait_random

This takes in 2 arguments: min, max

These arguments would be your wait time!

wait_random 0.0 0.5 input attack

Looping

Looping means to execute a command line N times. The N is specified during the first argument, whereas the command line the third.

loop 5 say /give @s diamond

This would loop executing the give command 5 times, resulting in giving yourself 5 diamonds.

Periodical Looping

This also means looping with a period. A period in physics normally means how long it takes to do one repitition of something, and that can be applied to this command as well.

This command would have 3 arguments

  • how many times it'll loop
  • the period
  • the command line
loop_period 5 1000 say /give @s diamond

This would essentially give the player a diamond every 1000 milliseconds, or one second, for 5 times.
