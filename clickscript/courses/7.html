Player Input 2

Learn how to rotate the client's camera slowly to a desired destination.

Turning the player's camera.

This command's name is turn_to, and has 4 modes

  • nearest_entity (specified ID)
  • nearest_block (specified ID)
  • any_entity
  • any_block

The parenthesized part is where you specify a minecraft ID

If the argument you've desided to use does not but a parenthesized part, you do not need to specify this extra argument.

Minecraft ID arguments were mentioned in the previous lesson. If you do not know this, we'd recommend not skipping lessons.

turn_to nearest_entity :player

This turns to the nearest player.

turn_to nearest_block #diamond_ore

This turns to the nearest diamond ore.

turn_to any_block

This turns to the nearest block.

turn_to any_entity

This turns to the nearest entity.

The after effect

Because this command does not snap instantly, and takes time to rotate your head, we've supplied scripters with an extra argument then.

This argument allows the command to run another command line after the rotation is finished.

turn_to nearest_entity :player then input attack

This turns to the nearest player then attacks it.
