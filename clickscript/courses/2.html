Basics

Learn basic script syntax: printing in console, checking errors in logs, commenting, and more!

Printing in console

Printing in console is fairly easy. The command name is print followed by a message of your choice.

print Hello World!

Checking errors

The console has more to just printing stuff in it. First of all lets talk about navigating to it.

Minecraft has a folder under .minecraft named logs.

Inside the folder, there would be a file you have to open, named latest.log

This is where everything from the console is saved to. From there, you can check any script errors that may have been caught and printed.

Commenting

Comments are made just like in Java, however they require to be on their on line. Comments are prefixed with 2 forward slashes.

// this is a comment
