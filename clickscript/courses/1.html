Getting Started

Get started on scripting basics: accessing your .minecraft folder, creating script files, and reloading scripts!

Opening .minecraft

To access your .minecraft folder, first open up your Windows search bar. Type in %Appdata%, then hit enter.

The .minecraft folder should be right there.

ClickCrystals data folder

The ClickCrystals' data folder is named ClickCrystalsClient. This folder should be visible when the mod is installed and ran. The ClickCrystals folder contains all the data that the mod has, including the client's config.

Scripts folder

The scripts folder contains all the script files (file extention .ccs). This folder is located inside of the ClickCrystals data folder.

Creating script files

The file name extensions for all script files should be .ccs, and nothing else. If the file name extention is .txt or .pdf or anything else, they would be skipped during execution.

You can check if the file name extention, or file type, is a CCS file. If it says .txt or anything else, be sure to change it.

Script files can be created by first creating a text document, then renaming it to .ccs

Reloading scripts

Scripts can be reloaded in game by typing the command

,ccs reload-scripts

This means you do not have to restart your game.

In-game test compiling

You test script compilation using the command.

,ccs compile ...

The ... can be replaced with any CCS script command.
