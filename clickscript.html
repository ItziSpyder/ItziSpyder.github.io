Create modules for ClickCrystals in under 5 lines!

  • Don't need to know how to code
  • Easy script reloading
  • Easy to understand and learn
  • Minimal syntax
  • Datapack-based idea!

module create potion-swap
description Switches to potions when you right click sword

on right_click if holding #sword if hotbar_has #splash_potion switch #splash_potion then input use then switch back

Simple, understandable syntax
